M-VAVE SMK-37 PRO

Important note:

For the script to work, you need to set the KeyPad to Preset 2 by pressing "Knob Bank" and "Fader Bank" simoulteniosly.
Confirm this by checking screen (3)
WHEELS (2)
Pitch Wheel Scroll the pitch Wheel up and down to control the pitch bend, the center position is the original pitch of the sound, lift the finger up will release the sound to the original pitch.
Modulation Wheel Scroll the modulation Wheel send continuous Midi CC message.
BUTTON AREA (4)
E 1-8 Track Pan.
K A1-A8 Track Send 1.
K B1-B8 Track Send 2.
F 1-8 Track Volume.
S C7 / C8 Scroll Track Bank up/down.
SCENE 2/DEVICE MODE
E 1-8 Device: Macros.
K A1-A8 Device: Common parameters.
K B1-B8 Device: User parameters.
F 1-8 Device: Envelope parameters.
S C1
+ Shift		 Device: Next user parameter page.
S C2 + C3
+ Shift		 Select previous/next cursor device.
S C7 + C8
+ Shift		 Select previous/next cursor track.
